## Description

Cross-platform mobile application for **Fucsol**, an e-commerce platform dedicated to natural products. This project is built with **Expo (React Native)** to ensure a unified codebase for both iOS and Android, prioritizing performance and developer experience.

## Tech Stack

* **Framework:** React Native (via Expo SDK)
* **Language:** TypeScript
* **State Management:** Zustand
* **Navigation:** Expo Router (planned) / React Navigation
* **Package Manager:** npm

## Getting Started

### Prerequisites

* Node.js (LTS version recommended)
* Expo Go app installed on your mobile device (iOS/Android) or an Emulator.

### Installation

\`\`\`bash
# 1. Clone the repository
$ git clone https://github.com/TU_USUARIO/fucsol-app.git

# 2. Navigate to the project directory
$ cd fucsol-app

# 3. Install dependencies
$ npm install
\`\`\`

## Running the App

\`\`\`bash
# Start the development server
$ npx expo start
\`\`\`

Scan the QR code with your phone (using the Expo Go app) or press `a` for Android Emulator / `i` for iOS Simulator.
